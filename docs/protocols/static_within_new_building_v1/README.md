# 新主楼静态同环境 Session-Holdout v1

目的：在环境不变化的前提下，评估设备级静态欺骗检测对未见完整 Session 的泛化能力。

| Split | Session | 场景 | 设备数 | 信号行数 | 正样本行数 |
|---|---|---|---:|---:|---:|
| train | 2025.07.29.20.16_新主楼 | st_L5 | 5 | 312,272 | 26,046 |
| train | 2025.07.29.18.42_新主楼 | st_L_15 | 3 | 150,132 | 57,529 |
| val | 2025.07.29.20.36_新主楼 | st_L5 | 1 | 50,312 | 2,642 |
| test | 2025.07.29.19.22_新主楼 | st_L1 | 4 | 250,321 | 76,993 |

所有集合均属于 `new_building`，每个完整录制 Session 只进入一个 split。该协议仍存在频段场景和设备组成变化，因此用于同环境开发诊断，不替代最终盲测。

固定首轮基线：27 维设备统计特征、因果窗口 `L=30`、LightGBM 默认阈值 0.5。
