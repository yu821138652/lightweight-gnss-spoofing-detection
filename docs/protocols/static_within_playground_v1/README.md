# 操场静态同环境 Session-Holdout v1

目的：在操场环境内，以未见完整静态 Session 测试设备级检测的泛化能力。

| Split | Session | 场景 | 设备数 | 信号行数 | 正样本行数 |
|---|---|---|---:|---:|---:|
| train | 2025.07.30.09.41_2025.07.30.09.45 | st_L5 | 6 | 56,629 | 0 |
| train | 2025.07.30.07.30_2025.07.30.08.01 | st_L_15 | 6 | 499,998 | 185,568 |
| val | 2025.07.30.09.48_2025.07.30.10.14 | st_L5 | 6 | 316,105 | 38,078 |
| test | 2025.07.30.08.40_2025.07.30.09.12 | st_L1 | 6 | 503,515 | 152,994 |

所有集合均属于 `playground`，且完整录制 Session 严格隔离。该协议仍含频段场景变化和设备日志变化，用于同环境开发诊断，不替代最终盲测。

固定首轮基线：27 维设备统计特征、因果窗口 `L=30`、LightGBM 默认阈值 0.5。
